へびゲーム

へびゲームとは

へびゲームは、へび（緑色のブロック）を操作して、盤上にランダムに発生するえさを与えて長くしていくゲームです。へびが外壁やへび自身の体に衝突すると、ゲーム終了です。

操作方法

へびは上下左右に動くことができますが、その場でUターンすることはできません。進む方向は上下左右の矢印キーで指定します。

なんでそんなに説明短いの？

はい。このゲームに関しては、特に取り上げるべき歴史があるわけでもなければ、興味深い性質があるわけでもないのです。そのため、ここの説明は極めて短いものになっています。それで物足りないという方のために、空白文字でも置いておきます。UnicodeのU+2001ってやつですね。
